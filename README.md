# Timbre_Escolar
Sistema de un timbre escolar automatizado basado en Arduino, utilizando RTC y un Relé para la activación del timbre

## Tecnologías Utilizadas
- Arduino UNO
- RTC DS3231
- LCD 16x2
- Modulo Relé
- C++ (Arduino)

## Cómo funciona: 
Arduino lee la hora actual del módulo RTC y la compara con los horarios almacenados. Al coincidir, se activa un relé para que suene la campana.
